# Neural Net XOR

Нейросеть прямого распростанения апроксимирует операцию XOR. На вход подаем два бита, на выходе получаем результирующий бит. Сеть снаписана на языке С++. Содержит 6 полноценных нейрона + 3 нейрона смещения.
